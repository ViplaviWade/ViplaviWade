*** Create a Node.js Project ***
npm init

*** Express JS is the framework of Node.js ***
npm install express 

*** Nodemon is used to  automatically restart Node application ***
npm install nodemon

*** Mongoose is an ODM Object Data Modelling library for MongoDB and Node.js ***
npm install mongoose

*** dotenv allows you to create env variables in .env file ***
npm install dotenv

*** body-parser is an npm module use to process data sent in an HTTP request body ***
*** Provides 4 express middleware for parsing JSON, Text, URL-encoded, raw data ***
npm install body-parser

*** Install the joi packet for validations ***
npm install joi

*** Install encryption/decryption library ***
npm install bcryptjs

*** Open standard to share info between client and server (generating auth token) ***

npm install jsonwebtoken
